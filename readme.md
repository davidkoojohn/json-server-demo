# json-server-demo

> This is website api demo powered by json-server.

## Project setup

### Install dependencies
```
yarn || npm install
```

### Compiles and hot-reloads for development
```
yarn start || npm run start
```

### Started db api
```
yarn db || npm run db
```

### Compiles and minifies for production
```
yarn build || npm run build
```
